  ### Dit ging goed/Heb ik geleerd: 
  Korte omschrijving met plaatjes
  Wat ik heb geleerd is het gebruik van de grid. Ook beheers ik beter met CSS, wat ik voor dit vak niet goed onder de knie had. Een voorbeeld is dat ik eerst ging positioneren met padding en margin door bijvoorbeeld:      "margin: -18em" wat er wel leuk uitzag op een specifiek formaat maar om het doel was om het responsive te maken wat natuurlijk niet werkt. Wat ik ook heb geleerd is om 1 specifieke sectie te selecteren doormiddel van nth-of-type. Ik vond het moeilijk om niet divjes, classes etc te gebruiken dus het was wel erg leerzaam om op zo'n manier iets te selecteren in CSS. Je moest ook denken aan toegankelijkheid, wat ik hiervoor niet wist wat er viel onder dat onderwerp. Het light en dark mode en de screenreader was niet moeilijk om toe te passen, maar je moest er wel aan denken om het voor de gebruiker zo duidelijk mogelijk te maken. Ik wist eerst niet hoe je dat moest toepassen maar richting het einde van het project had ik meer lol in coderen dan eerst en leerde ik snel hoe ik het kan toepassen in mijn code.
